웹표준 마크업

  • 의미에 맞는 요소(태그)를 선택하여 사용, 시멘틱 마크업
  • 구조(html, 콘텐츠), 디자인(css), 동작(Javscript)의 파일분리
  • 시멘틱 태그를 적극적으로 활용
  • 콘텐츠에는 항상 제목을 달아주는것을 권장
  • 사용자 입력폼(폼문)은 form, fieldset, legend는 같이사용
  • 디자인적인 요소는 생략, 기능/콘텐츠(내용) 중심으로 작성
  • 링크는 a, 동작은 button
  • 유효성검사(html validator, 마크업 문법검사)는 필수
  • 의미가 없는 디자인용 태그 : div, span
  • 비권장 태그 : 디자인적인 의미를 가지는 태그(b, i, u, center, hr...), 프레임관련(framset, frame, iframe...)
