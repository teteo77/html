가상클래스(가상선택자)

가상클래스(가상선택자)

  • :first-child / :last-child : 형제번호가 1번째/마지막인 요소
  • :nth-child(n) : 형제번호가 n번째인 요소
  • :nth-of-type(n) : 형제중의 같은 태그!!!의 순번이 n번째인 요소
  • :link / :visited : a 태그의 링크주소가 방문하지 않은 경우/ 방문했던 주소인 경우 네이버 네이버
  • :hover / :active : 마우스 올렸을 때/ 마우스 눌렀을 때
  • :focus : 포커스(활성화) 되었을 때
  • :checked : 체크가 되었을 때 동의합니다. 나는 다른 span
  • :empty : 자식이 비어있을 때

가상요소

  • ::before : 선택자 앞부분에 가상의 태그를 삽입
  • ::after : 선택자 앞부분에 가상의 태그를 삽입
  • ::placeholder : input의 placeholder 선택자
  • 가상요소는 content 속성은 필수, 특정태그(input, img, video...)는 작동안함
