HTML5 내용작성 태그(요소, Element) h1 : 페이지 제목 h2~h6 : 소제목 p : 글, 문단, 문장 img : 이미지 video : 비디오 audio : 오디오 ul / ol / dl : 리스트 a : 링크, 네이버 table : 표 시멘틱 태그 시멘틱 태그란 의미를 담고 있는 태그 header : 홈페이지의 상단영역, 로고, 제목, 메뉴, 검색, 로그인, SNS... main : 홈페이지의 콘텐츠 영역 footer : 홈페이지 하단영역, 카피라잇, 사이트 이용에 필요한 정보, 연락처, 공지사항, 사이트맵... section : 큰 단위의 콘텐츠(정보) 그룹 article : 작은 단위의 콘텐츠 그룹 nav : 홈페이지 메뉴 aside : 홈페이지 사이트 영역, 보조콘텐츠(날씨, 배너광고, 바로가기 아이콘....) address : 주소, 카피라잇.... strong/em : 강조 details : (토글용) 상세보기 폼문관련 태그 form : 사용자의 정보를 서버로 전송하는 기능의 태그 input : 사용자의 정보를 입력받는 태그 fieldset : 폼영역을 알려주는 기능 legend : 폼영역(fieldset)의 제목 button : (전송)버튼 label : input의 포커스(활성화)를 연결해주는 태그 textarea : 여러줄 입력시 성별 구분하기 남 여 보내기 네이버 http://naver.com 네이버 http://naver.com 네이버 http://naver.com 1 1 3 4 5 6 9 13 14
