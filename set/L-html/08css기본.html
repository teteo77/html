CSS 기본

CSS 작성방식

  • HTML 문서에서 디자인을 꾸미는 모든 활동
  • 인라인 방식 : 태그(요소)에 style 속성으로 작성하는 방식, 비권고 방식
  • 내부스타일 : style 태그로 작성하는 방식, 비권고 방식
  • 외부스타일 : link태그로 불러오는 방식, 웹표준 권고사항

CSS 선택자⭐⭐⭐⭐⭐

  • CSS 선택자란 스타일링 대상요소(태그) 또는 경로
  • * : 전체선택자
  • 태그(li) : 요소선택자
  • # : 아이디 선택자
  • . : 클래스 선택자
  • (빈칸) : 자손선택자
  • > : 자식선택자
  • :nth-child(숫자) : 형제번호가 (숫자)번째인 요소
  • [속성] : 속성선택자
  • + : 형제선택자
  • ~ : 선택자의 동생들을 선택
  • , : 그룹선택자

CSS에서의 색상표현

  • 색상명 : red
  • 웹색상코드(16진수) : #FF000, #f00, 0~9+a~f
  • RGB : rgb(255,0,0)
  • RGBa : rgba(255,0,0,1)

CSS 단위

  • pt, cm, mm... : 인쇄용, 비권고
  • px : 픽셀(모니터의 점한개), html 문서의 표준단위
  • % : (부모 요소에 대한) 배율단위
  • rem : 루트(html, body)가 가진 글자크기에 대한 배율단위
  • em : 부모가 가진 글자크기에 대한 배율단위
  • 숫자 : x배
  • vh, vw : 뷰포트의 배율단위(%단위와 비슷)
