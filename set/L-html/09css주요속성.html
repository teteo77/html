CSS 주요속성

외곽선(border)

  • border: 1px solid red; // 선두께, 선타입(solid, dotted, dashed), 색상
  • border-color : green; // 선색
  • border-radius : 10px // 둥근모서리
  • border-top/bottom/left/right: 2px solid green // 한쪽방향에 대한 선설정

글자(font)

  • color : red // 글자색
  • font-size : 20px // 글자크기
  • font-weight : bold // 글자 굵기(100~900)
  • font-style : italic // 이탤릭체 설정(italic, normal)
  • font-family : 궁서체 // 폰트(서체) 지정, 웹표준에서 최소 2개이상 지정을 권고
  • line-height : 100px
    // 줄간격
  • font : bold italic 20px/20px 궁서체, serif // 폰트 대표속성
  • text-indent : 50px // 들여쓰기, 내어쓰기
  • letter-spacing: 10px // 자간설정
  • text-align : center // 글자정렬(left, center, right, justify)
  • user-select : none // 선택(드래그)방지
  • pointer-events : none // 이벤트(클릭, 드래그, 선택, 작동...) 방지
  • text-shadow : 2px 2px 4px #000 // 글자 그림자 효과
  • text-decoration : underline // 밑줄설정(none)

배경

  • background-image : url(1.jpg) // 배경이미지 지정
  • background-repeat : no-repeat // 배경이미지 반복설정
  • background-size : 100px, 100%, cover, contain // 배경사이즈
  • background-position : center center, 50px 30% // 배경위치
  • backgound-color : red // 배경색 지정
  • background : none, url(01.jpg) no-repeat center right / 50px 50px // 배경 대표속성
  • background-attachment: fixed; // 배경이미지 고정

크기/여백

  • width / height : 100px, 100% // 폭, 높이
  • min-width / min-height : 400px // 최솟값
  • max-width / max-height : 400px // 최댓값
  • margin / padding : 20px // 선바같여백, 선안쪽여백
  • overflow : hidden // 넘치는 콘텐츠(내용)의 보임여부 설정

display 속성

  • inline : 인라인, 글자단위의 태그(a, strong, span, em...) // 크기지정 X, 여백 부분적으로 가능
  • block :
    블록
    , 박스단위의 태그(ul, li, h1~h6, form, div...) // 한줄에 한개, 크기지정 O, 여백 O
  • inline-block : 인라인블록, 크기지정 O, 여백지정 O, 글자정렬(img, table, input, video...)
  • none : 화면에 출력하지 않음 나는 안보임

position 속성

  • static : 기본값(레이어 없이 바닥(body)에 놓여진 상태)
  • absolute : 레이어 개념으로 다른거의 영향을 주지 않고 개별적인 위치지정absolute
  • fixed : 레이어개념으로 화면(고정)좌표로 위치지정 fixed
  • relative : 레이어 속성을 주지 않고 자손의 좌표의 기준이 되게 지정 absolute
