IR/IS 기법

IR(Image Replacement), 이미지 대치기법

  • IR기법이란 마크업(html)은 텍스트로 하고 CSS에서는 이미지로 교체하는 기법
  • 마크업 : 오늘의 날씨 : 맑음
  • CSS : 오늘의 날씨 : 맑음

IS(Image Sprite)기법

  • 과거에는 이미지용량을 줄이는게 최적화였지만 현재에는 용량보다 갯수를 줄이는게 최적화에 유리
  • 날씨범례
    • 맑음(낮)
    • 맑음(밤)
    • 구름조금(낮)
    • 구름조금(밤)
    • 구름많음(낮)
    • 구름많음(밤)
    • 흐림
    • 약한비
    • 비
    • 강한비
    • 약한눈
