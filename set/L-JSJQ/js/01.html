자바스크립트 시작

자바스크립트 특징

  • 브라우저에서 실행되는 세계에서 가장 많이 사용되는 언어
  • 자바(Java/백엔드)와 다른 클라이언트(프런트엔드) 언어
  • 컴파일 과정이 없는 인터프린터 언어
  • 소스가 노출, 원본에 대한 저작권 권리 행사 힘들.
  • 플랫폼의 영향이 적고 운영체제의 제한을 받지 않음.
  • Pure scripte, 순수 스크립트, 바닐라 스크립트는 라이브러리를 사용하지 않는 스크립트 말함.
  • 대소문자를 구분함.
  • 큰따옴표("), 작은따옴표(') 겹침주의

자바스크립트 작성방법

  • 인라인 스크립트 : 태그의 (이벤트) 속성으로 작성, 웹표준에서의 비권고사항
  • 내부스크립트 : script요소(태그)로 작성, head/body 안쪽에 작성 가능 하지만 body의 마지막 내용으로 작성을 권고.
  • 외부스크립트 : script요소로 불러오는 방식, 내부스크립트는 html의 script태그 내부에 작성
  • 주석 : // 한줄주석, 말꼬리 주석,

변수

  • 변수란 변하는 수/값/데이터/문자...등을 담는 메모리 공간
  • 변수의 선언 : var(함수단위), let(스코프단위), const(값을 넣으면 변경 불가능)
  • 변수의 데이터형 : 숫자, 문자, 논리형(true, false), null, undefined
  • 변수 네이밍 규칙
    • 첫자로 숫자가 올수없다.
    • 특수기호, 빈칸, (한글)은 사용할 수 없고 영문+숫자로 조합
    • 기본적으로 소문자만 사용(이유있는 경우 제외)
    • 의미에 맞는 이름 사용
    • super_man : 언더바표기법:(snakecase)
    • superMan : 낙타표기법(camelcase)
    • 같은 이름으로 중복 안됨
