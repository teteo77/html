BOM(Browser Object Model)

  • 브라우저에서 제공하는 객체모델
  • window : open(), alert(), prompt(), cconfirm(), setInterval() : 반복실행, setTimeout() : 지연실행
  • screen : width, height, colorDepth
  • location : href, hash, host, hostname, protocol
  • history : back(), foward()
  • navigator : appCodeName, appName, language, product, paltform, userAgent
