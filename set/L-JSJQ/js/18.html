이벤트와 애니메이션

이벤트

  • click() / dbclick() : 클릭 / 더블클릭 했을 때
  • load() : 선택자의(html을 포함한) 자손을 포함한 외부소스가 로딩이 되었을 때
  • ready() : 선택자의 html 태극 로딩되었을 때
  • mouseover() / mouseout() : 마우스 올렸을 때/벗어났을 때
  • mousemove() / mousedown() / mouseup() : 마우스 움직일때 / 눌렀을 때 / 때였을 때
  • keydown() / keyup() : 키눌렀을 때 / 떼었을 때
  • focus() / blur() : 포커스 얻었을 때 / 잃었을 때
  • scroll() : 스크롤 했을 때
  • change() : value값이 변경되었을 때
  • resize() : window 사이즈가 변경 되었을 때
  • on() : 이벤트리스너(이벤트 감시자), 여러개의 이벤트를 걸거나, 동적요소에 이벤트를 걸때, 근래 나온 최신 이벤트를 걸때

효과 애니메이션

  • hide() / show() / toggle() : 숨김 / 보임 / 토글
  • fadeIn() / fadeOut() / fadeToggle() : 부드럽게 보임
  • slideUp() / slideDown() / slideToggle() : 위로 가려짐 / 아래로 보임 / 토글
  • animate()
