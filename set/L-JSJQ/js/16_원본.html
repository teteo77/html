jQuery 기본

  • 제이쿼리(jQuery)는 자바스크립트로 만든 라이브러리(프레임워크: 리액트, 뷰, 앵글러...)
  • releases.jquery.com
  • 버전 1.x(구 브라우저 호환), 버전 2.x(구 브라우저 호환 제거, 가벼워짐), 3.x(구 브라우저 호환 안됨, 기능 강화)
  • uncompressed(원본), minified(압축), slim(간소화 버전), slim minified(간소화+압축)
  • 선택자를 jQuery면 명령어도 jQuery 여야함.

기본 메서드

  • $(), jQuery() : jQuery식 선택자
  • css() : style 속성을 변경/가져올 때
  • attr() : 태그의 속성을 변경/가져올 때
  • text() : 글자를 변경/가져올 때
  • html() : 글자+html을 변경/가져올 때
  • addClass() / removeClass() / toggleClass() : 클래스 추가/ 삭제/ 토글
  • eq() : jquery 선택자의 인덱스번호로 재선택
  • get(0) : jquery 선택자에 자바스크립트 명령어를 사용할 경우
  • click() : 클릭이벤트
  • val() : value 속성값을 변경/ 가져올때

수치관련

  • width() / height() : 폭/높이값을 적용, 가져올 때
  • offset().top / offset().left
  • scrollTop() : 스크롤값을 얻거나 적용

객체 편집 메서드

  • before() / after() : 선택자 앞/뒤에 추가
  • prepend() / append() : 선택자 자식 맨앞/맨뒤로 추가
  • clone() : 선택자를 복사
  • remove() : 선택자를 삭제
  • empty() : 선택자 안의 내용(자손)을 비움
원본

기타 메서드

  • parent() : 선택자의 부모를 선택
  • parents(".on") : 직계조상중의 클래스 on을 선택
  • children("span") : 선택자 자식중의 span만 선택
  • find(".on") : 선택자 자손중에 클래스 on을 선택
  • next() / prev() : 선택자 다음/이전(형제)요소 선택
  • each() : 선택자 개별적으로 실행
