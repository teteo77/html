  • classList.remove() : 클래스 제거
  • classList.toggle() : 클래스 토글

  • window.scrollY : 현제ㅐ스크롤값 얻기
  • window.scrollTo(0, 500) : 스크롤 이동
  • window.scrollTo(0, 500) : 스크롤 이동
  • window.scrollTo({left:0, top:1000, behavior:"smooth"})
