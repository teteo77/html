DOM(Document Object Model)

  • HTML 문서의 구조와 그 정보를 말함.
  • 선택자.명령/속성, 예) document.body.style
  • getElementById("green") : 아이디 선택자
  • getElementsByClassName("blue") : 클래스 선택자
  • getElementsByTagName("li")[0] : 태그 선택자

기본속성/명령어

  • 선택자.style.스타일속성 : DOM(태그)의 디자인을 바꿀때
  • 선택자.속성 : 태그의 속성을 가져오거나 변경할 때
  • innerText : 글자를 얻거나 변경
  • innerHTML : 글자+태그를 얻거나 변경
