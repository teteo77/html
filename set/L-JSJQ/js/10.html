이벤트(Event)

  • 이벤트란 방문자가 입장이후의 모든 움직임(상호작용)
  • onclick : 클릭했을 때
  • ondblclick : 더블클릭 했을때
  • onmouseover : 마우스 올렸을 때
  • onmouseout : 마우스 벗어났을 때
  • onkeypress : 키를 눌렀을 때 "키누름", 특수키 안됨, 대소문자를 구분
  • onkeydown : 키를 눌렀을 때 "키누름", 특수키 기능, 모두 대문자로 인식, 방향키 인식
  • onkeyup : 키를 (눌렀다) 떼었을 때
  • onfocus : 포커스를 얻었을 때
  • onblur : 포커스를 잃었을 때
  • onchange : value 값이 변경되었을 때
  • onload : 하위요소가 모두 로딩되었을 때
  • onresize : window(브라우저 크기)의 크기가 변경되었을 때
  • this 키워드 : 함수, 이벤트안에서 사용되는 이벤트 발생 요소(태그)를 가르키는 예약어
  • return 키워드: 함수, 이벤트를 종료하고 값을 반환 또는 이벤트 중단(return false)
  • addEventListener : 이벤트를 중첩, 다중이벤트, 동적요소/ 최신 이벤트에 가능
  • 화살표 함수()
