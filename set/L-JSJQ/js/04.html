객체(Object)

  • 기능(Method)과 속성(Property)을 가지는 단위
  • 객체.Method() : 객체안에 있는 명령어를 실행, document.write()
  • 객체.Property : 객체안의 정보를 가져올때, document.body : getter
  • 객체.porperty = 값 : 객체안의 정보를 설정할 때, document.body.style.color = 'red' : setter
  • 내장객체 : 문자, 날짜, 숫자, 배열, 정규배열(정규식)

실습예제

  • 객체의 생성
  • 날짜객체 - 오늘 날짜
  • 날짜객체 - 디데이(기념일) 계산
  • 숫자관련 객체
  • 배열 기본 사용법
  • 문자객체
  • 정규식 기본 사용법
