연산자

  • 산술연산자 : +,-,*,/,%(나머지)
  • 문자결합 : + (어느 한쪽이라도 문자일 경우)
  • 대입연산자 : =, +=, -=, *=, /=, %=
  • 비교연산자 : ==(===), >, <, >=, <=, !=(!==)
  • 논리연산자 : &&(그리고), ||(또는), !(아니면)
  • 증감연산자 : 변수++, 변수--, ++변수, --변수
  • 삼항연산자 : 조건식 ? 참일때 실행문 : 거짓일때 실행문
